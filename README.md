Juego de robots hecho en Java Springboot con Maven y Mongodb como base.

Endpoints:

```
POST /robots/create
```


body json
```
{
    "name":"test",
    "setup":"test"
}
```


```
POST /teams/create
```


body json:
```
{
    "name":"Epsilon Delta Limit",
    "budget":10000000
}
```

```
/teams/show
```
